# Milestone Template

Milestones SHOULD be described in a table format with the following columns:
- `#` – Sequential identifier.
- `Name` – Descriptive title of the milestone.
- `Target Date` – ISO‑8601 date or relative due date.
- `Status` – `Planned`, `In Progress`, `Completed`, or `Blocked`.

Example:
```markdown
| # | Name                      | Target Date | Status       |
|---|---------------------------|-------------|--------------|
| 1 | Architecture Review      | 2026-08-01  | Completed    |
| 2 | Core Library Implementation | 2026-09-15  | In Progress |
```
